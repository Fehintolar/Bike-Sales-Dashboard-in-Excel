# Bike-Sales-Dashboard-in-Excel
### An interactive bike sales dashboard built in Microsoft Excel to analyze customer demographics and identify factors influencing bike purchases.

## Project Objective
### To analyze bike customer data and uncover insights that can support better business decisions around customer targeting and sales strategy.

## Dataset
## Source: [GitHub Dataset from Alex The Analyst](http://excel-tutorial/Excel%20Project%20Dataset.xlsx%20at%20main%20%C2%B7%20AlexTheAnalyst/Excel-Tutorial%20(github.com))

- 1,026 records
* 13 columns

## Tools Used
- Microsoft Excel
- Pivot Tables
+ Pivot Charts
+ Slicers
+ Dashboard Designs

## Data Cleaning
- Removed 26 duplicates
- Standardized gender and marital status labels
+ Formatted income column
* Created customer age brackets

## Dashboard Features
The dashboard analyzes:

Average income per purchase
+ Commute distance
+ Age bracket
* Customer region
* Purchase by gender
- Purchase by marital status

Interactive filters included:
- Education
- Occupation
  
## Key Insights
- Customers with higher income are more likely to purchase bikes
- Middle-aged customers are the strongest buyer segment
- Short-distance commuters show stronger bike purchase behavior
- North America contributes the largest customer share
- Married customers slightly outnumber single customers in purchases

## Business Recommendations
- Focus marketing on middle-aged professionals
- Promote bikes as short-distance commuting solutions
- Prioritize high-performing regions
- Use demographic segmentation for more targeted campaigns
